Duplicate/Copy any and all post types, including taxonomies & custom fields with a click!

== Description ==

This plugin allows users to clone posts of any post type, or copy them to new drafts for further editing.

This plugin was created to make an exact duplicate/copy of a selected post. Custom post types are supported, along with custom taxonomies and custom fields.

This plugin is simply used to quickly and easily duplicate a post. Just hover over a post in the edit screen and click on  'Duplicate' to create a duplicate post.
	
== Installation ==
To install Quick Post Duplicator using the built-in plugin installer:

	1. Go to Plugins > Add New.
	2. Type in Quick Post Duplicator in Search Plugins box or click a tag link below the screen.
	3. Find the Quick Post Duplicator Plugin in the list.
	4. Click Install Now to install the WordPress Plugin.


To install Quick Post Duplicator manually:

	1. Download your WordPress Plugin to your desktop.
	2. If downloaded as a zip archive, extract the Plugin folder to your desktop.
	3. With your FTP program, upload the Plugin folder to the wp-content/plugins folder in your WordPress directory online.
	4. Go to Plugins screen and find the Quick Post Duplicator Plugin in the list.
	5. Click Activate to activate it.


== Frequently Asked Questions ==

= Does this plugin support custom post types? =

Yes, all registered custom post types are supported.

= How to duplicate custom post types? =

Go to wp-admin >> pages/posts/custom post , On quick edit bar click on duplicate.
